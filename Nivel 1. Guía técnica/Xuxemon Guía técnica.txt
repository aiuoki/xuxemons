# Guía Técnica de Xuxemon

Esta guía tiene como objetivo explicar cómo abrir la página web de Xuxemon en cualquier ordenador con sistema operativo Windows. A continuación, detallaremos los requisitos y pasos necesarios para acceder al código.

## Requisitos Previos

1. **Conexión a Internet:** Necesitarás una conexión activa para descargar el código.
2. **Visual Studio:** Asegúrate de tener instalado Visual Studio para ejecutar el código.
3. **Angular:** Instala Angular en tu ordenador utilizando el siguiente comando:

     
    npm install -g @angular/cli
  

    Esto instalará globalmente la herramienta angular-cli.

4. **Laravel para el Backend:** Utilizaremos Laravel, por lo que también necesitarás una conexión a una base de datos. Para simplificar este proceso, instalaremos XAMPP, un conjunto de servidores web multiplataforma diseñado para trabajar con PHP.

    Accede a la página oficial de XAMPP para descargar e instalar: [XAMPP](https://www.apachefriends.org/es/index.html)

5. **Composer:** Descarga Composer, una herramienta para la gestión de dependencias en PHP, desde: [Composer](https://getcomposer.org/)

## Configuración y Ejecución

1. **Visual Studio - Laravel:**
   - Abre Visual Studio y accede a la carpeta llamada "Main".
   - Utiliza la terminal proporcionada por Visual Studio.
   - Ejecuta el siguiente comando:

      
     cd Laravel
   

   - Asegúrate de tener Apache y MySQL activos en XAMPP.
   - Inicia la parte del backend con:

      
     php artisan serve
   

2. **Angular:**
   - Abre otra terminal.
   - Accede al directorio de Angular con:

      
     cd Angular
   

   - Inicia la parte visual con:

      
     ng serve -o
   

Con estos pasos, habrás configurado y ejecutado correctamente la aplicación Xuxemon.
